Cloudflare API 토큰 생성

다음 단계를 따라 API 토큰을 생성하세요:

  1. Cloudflare Dashboard → "My Profile" → "API Tokens"
  2. "Create Token" → "Custom token" 선택
  3. 권한 설정:
    • Account - Cloudflare D1:Edit
    • Zone - Zone:Read, Zone Settings:Edit
    • Zone Resources - Include All zones
  4. "Continue to summary" → "Create Token"
중요: 생성된 토큰은 안전한 곳에 저장하고, 다른 사람과 공유하지 마세요.

D1 데이터베이스 생성

Cloudflare 대시보드에서 D1 데이터베이스를 생성하세요:

  1. Cloudflare Dashboard → "Workers & Pages" → "D1 SQL Database"
  2. "Create database" 클릭
  3. Database name: teamk-data
  4. "Create" 클릭
  5. 생성된 데이터베이스의 "Database ID" 복사

Cloudflare Worker 배포

API 엔드포인트를 제공할 Worker를 배포하세요:

  1. Cloudflare Dashboard → "Workers & Pages" → "Create application"
  2. "Create Worker" → Name: teamk-api
  3. 아래 코드를 Worker 에디터에 붙여넣기
  4. "Save and Deploy" 클릭
  5. Settings → Variables → "D1 Database Bindings" 추가:
    • Variable name: TEAMK_DB
    • D1 database: 위에서 생성한 teamk-data 선택
