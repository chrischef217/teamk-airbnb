Firebase 데이터베이스 설정

PC와 모바일 간 실시간 데이터 동기화를 위해 Google Firebase를 사용합니다.

✨ 완전 무료이고 설정이 매우 쉽습니다!

현재 연결 상태

확인 중...
1

Firebase 프로젝트 생성 (2분 소요)

매우 간단합니다:

  1. Firebase Console 접속
  2. "프로젝트 추가" 클릭
  3. 프로젝트 이름: teamk-data
  4. Google Analytics: 사용 안 함 선택
  5. "프로젝트 만들기" 클릭
Firebase Console 열기
2

Realtime Database 활성화 (1분 소요)

  1. Firebase 프로젝트에서 "Realtime Database" 클릭
  2. "데이터베이스 만들기" 클릭
  3. 위치: "asia-southeast1" 선택 (싱가포르 - 가장 가까움)
  4. 보안 규칙: "테스트 모드에서 시작" 선택
  5. "사용 설정" 클릭
중요: "테스트 모드"를 선택하면 30일간 누구나 읽기/쓰기 가능합니다. 나중에 보안 규칙을 수정할 수 있습니다.
3

API 키 복사 (30초 소요)

  1. Firebase 프로젝트 ⚙️ 설정 (톱니바퀴) → "프로젝트 설정"
  2. "일반" 탭에서 아래로 스크롤
  3. "웹 API 키" 복사
  4. "프로젝트 ID" 복사
4

Firebase 연결 설정

Firebase 프로젝트 설정에서 복사한 Web API 키

Firebase 프로젝트 ID

완전 무료: Firebase Realtime Database는 월 1GB까지 무료입니다.
일반적인 사용으로는 절대 초과하지 않습니다.

설정 완료 후 할 일

  1. 데이터 마이그레이션: 기존 localStorage 데이터를 Firebase로 이전
  2. PC/모바일 테스트: 투자자 추가 후 다른 기기에서 확인
  3. 실시간 동기화 확인: 한 기기에서 수정하면 다른 기기에서 즉시 반영
